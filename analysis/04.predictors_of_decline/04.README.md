# 04. Predictors of Decline

## Overview

This directory contains analyses used to identify **environmental
predictors of population decline** in Bank Swallow. Using ESU
delineations generated in `03.delineate_ESUs`, this workflow combines
abundance trend data with climate and environmental predictors to model
drivers of decline across conservation units.

This workflow includes:

-   Preparing ESU-level spatial datasets
-   Extracting climate and environmental predictors
-   Visualizing ESU-level patterns
-   Fitting ESU-level GAMs
-   Comparing predictors across grouping schemes

The outputs from this directory identify environmental variables
associated with population decline across ESUs.

## scratch/

Intermediate files and spatial datasets used for modeling.

These files are reproducible and typically large, so they are not
tracked in GitHub.

### spatial_predictors/

Contains spatial datasets used for modeling:

-   ESU assignment per raster cell
-   Abundance trend values per cell
-   Climate change variables
-   Environmental predictor datasets

These datasets are merged to create the final modeling input.

------------------------------------------------------------------------

### delta_climate/

Contains processed climate change variables used as predictors,
typically derived from TerraClimate.

These may include:

-   Temperature change
-   Precipitation change
-   Soil moisture change
-   Wind speed change

------------------------------------------------------------------------

### TerraClimate/

Raw or processed TerraClimate data used to generate environmental
predictors.

------------------------------------------------------------------------

### genoscape_stars/

Rasterized ESU assignment surfaces used for spatial modeling.

These rasters correspond to:

-   G4 ESUs
-   G5 ESUs
-   GM5 ESUs
-   GM6 ESUs

These are used to assign ESU identity to abundance trend grid cells.

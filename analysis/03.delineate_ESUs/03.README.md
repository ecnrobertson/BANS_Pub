# 03. Delineate ESUs

## Overview

This directory contains code and outputs for delineating **Evolutionarily Significant Units (ESUs)** in Bank Swallow using population genomic structure and admixture analyses.

This workflow includes:

-   Population structure analysis (PCA + admixture)
-   Assignment of individuals to ESUs
-   Creation of genoscape surfaces
-   Delineation of ESU polygons
-   Visualization of ESUs across multiple grouping schemes

The outputs from this directory define ESUs used in downstream analyses of population trends, environmental drivers, and genomic offset.

## ESU Grouping Schemes

This directory produces four ESU delineation schemes:

| Scheme | Description                  |
|--------|------------------------------|
| G4     | Genetics-only, 4 ESUs        |
| G5     | Genetics-only, 5 ESUs        |
| GM5    | Genetics + migration, 5 ESUs |
| GM6    | Genetics + migration, 6 ESUs |

These alternative schemes allow evaluation of how ESU delineation influences:

-   Trend interpretation
-   Environmental drivers
-   Conservation prioritization
